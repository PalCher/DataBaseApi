
1. Консольное приложение, которое позволяет узнать цену товара по его имени, либо вывести сообщение «Такого товара нет», если товар отсутствует в базе. Консольная команда: "/цена товар545".

2. Есть возможность изменения цены товара (указываем имя товара и новую цену). Консольная команда: "/сменитьцену товар10 10000".

3. Вывести товары в заданном ценовом диапазоне. Консольная команда: "/товарыпоцене 100 600".
